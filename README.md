#### Routes
  - /theaters [GET]
  - /movies [GET]
  - /movies?is_showing [GET]
  - /movies/:movieId [GET]
  - /movies/:movieId/reviews [GET]
  - /movies/:movieId/theaters [GET]
  - /reviews/:reviewId [GET/PUT/DELETE]
